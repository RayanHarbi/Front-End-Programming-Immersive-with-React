# Exercises: JavaScript loops

## Print every number from 0 to 10

```
for (let i = 0; i < 10; i++) {
    console.log(i)
}
```

## Print every number from 10 to 0


```
for (let i = 10; i >= 0; i--) {
    console.log(i)
}
```


## Print every number from 4 to -16


```
for (let i = 4; i >= -16; i--) {
    console.log(i)
}
```


## Print every fifth number from 8 to 41


```
for (let i = 8; i < 42; i = i + 5) {
    console.log(i)
}
```
